# PET Strapping Tensile Strength Calculator

เครื่องมือเว็บแอปพลิเคชันระดับมืออาชีพสำหรับคำนวณค่า Tensile Strength และออกรายงานผลการทดสอบสำหรับ **สายรัดพลาสติก PET (PET Strapping)** โดยเฉพาะ

## ฟีเจอร์หลัก (Key Features)
- **Calculations:** คำนวณพื้นที่หน้าตัด (Area) และ Tensile Strength (MPa) อัตโนมัติ
- **Joint Efficiency:** รองรับการคำนวณประสิทธิภาพของจุดต่อแบบ Friction Weld และ Steel Seal
- **Professional Report:** สร้างรายงานที่สวยงามพร้อมสั่งพิมพ์หรือบันทึกเป็น PDF ได้ทันที
- **Premium UI:** ดีไซน์แบบ Modern Dark Mode พร้อม Glassmorphism สวยงามและใช้งานง่าย

## วิธีการใช้งาน (How to Use)
1. ป้อนข้อมูลชื่อวัสดุและ Batch No.
2. ระบุความกว้าง (Width) และความหนา (Thickness) ของสายรัด
3. ป้อนค่าแรงดึงขาด (Break Strength) และการยืดตัว (Elongation)
4. เลือกประเภทจุดต่อ (Joint Type)
5. กดปุ่ม **"พิมพ์รายงาน (PDF)"** เพื่อรับรายงานผลการทดสอบ
